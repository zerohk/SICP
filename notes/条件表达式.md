# if 语句

## 一般格式

```lisp
; 一般格式
(if <predict>
	<consequent> ; 为真时执行
	<alternative> ; 为假时执行
)
```

## 示例

```lisp
(define (positive? x)
  (if (> x 0)
      "正数"
      "非正数"))
```

```lisp
;; 判断一个数是否是偶数
(define (even? x)
		(if (= 0 (modulo x 2))
			"偶数"
			"奇数"))
```

# cond 表达式

## 一般格式

```lisp
(cond 
	(<条件1> <表达式1>)  ;; 表达式1为真时执行
	(<条件2> <表达式2>)  ;; 表达式2为真时执行
	...
	(<条件n> <表达式n>)
	(else <表达式>)      ;; 其他条件执行表达式
)
```

## 示例

```lisp
;; 判断一个数的符号
(define (sign x)
	(cond (> x 0) "正数"
	      (< x 0) "负数"
	      else "零"	  
         )
)
```

```lisp
#lang sicp
(define (grade x)
            (cond ((>= x 90) 'A)
                  ((>= x 80) 'B)
                  ((>= x 70) 'C)
                  ((>= x 60) 'D)
                  (else 'F)))
```
